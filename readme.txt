Простий та зручний плагін для підключення доставки популярною службою "Delivery" на ваш сайт WooCommerce.

== Description ==

Плагін використовує API Delivery для отримання свіжої інформації про населені пункти, міста та відділення, які покриває служба.

Для отримання данных йде GET запит до мого сайту API. https://api.ice-design.pp.ua/delivery.php

Вони публічні https://www.delivery-auto.com/api/v4/Public/ та запит до них йде при виборі пункту при оформленні замовлення службою доставки Delivery.

Дозвіл на їх використання https://www.delivery-auto.com/uk/Home/Api що зазначено в їх документації 
https://www.delivery-auto.com/userfs/LocalizableFiles/uk-UA/delivery-api/API_Delivery_v3.4.1.pdf?ver=3

Політика конфіденційності ТОВ «ДЕЛІВЕРІ»

== Переваги ==

* Проста та швидка установка.
* Підтримка найсвіжішої версії Woo Commerce.

== Відомі питання ==

* Плагін використовує API Delivery для отримання свіжої інформації про населені пункти, міста та відділення, які покриває служба.

== Installation ==

1. Upload the entire `delivery` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
